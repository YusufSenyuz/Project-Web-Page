Project Summary

Our project handles the problem of house renting. Rents are regulated by our system. Expected rental value of each building is calculated by the Machine Learning algorithm. Machine Learning algorithm checks the other rent increases in each area. In our project, deposits of each rental apartment will be taken by the government. The government will be able to earn money by using the collected deposits. With the implementation of the project, rent problems such as exorbitant price increases and tenant-landlord disputes will be solved.
